# ⚽ W杯2026 予想バトル

パリミュチュエル方式（競馬と同じ）の完全公平な予想バトルアプリ。

## 機能

- **3連単** — 1位・2位・3位を順番通りに予想（高配当）
- **3連複** — トップ3チームを順不同で予想（中配当）
- **単勝** — 優勝チームだけを予想（低リスク）
- **分割賭け** — 掛け金を複数の賭け種別に分配
- **ライブオッズ** — 参加者の予想に応じてリアルタイム変動
- **予想報酬表示** — 的中した場合の報酬をリアルタイム表示
- **管理者パネル** — 締め切り設定・結果入力・配当計算

## Renderへのデプロイ手順

1. GitHubにリポジトリを作成
   ```
   cd wc2026_bet
   git init
   git add .
   git commit -m "initial commit"
   git remote add origin https://github.com/あなた/wc2026-bet.git
   git push -u origin main
   ```

2. [render.com](https://render.com) にアクセスしてアカウント作成（無料）

3. "New" → "Blueprint" を選択 → GitHubリポジトリを接続

4. `render.yaml` が自動認識され、WebサービスとPostgreSQLが自動作成される

5. デプロイ完了後、表示されたURLを参加者20人に共有するだけ！

## ローカル起動（テスト用）

```bash
cd wc2026_bet
pip install -r requirements.txt
python app.py
```
→ http://localhost:5000 にアクセス

## 管理者操作

1. `/admin` にアクセス
2. 初期パスワード: `admin1234`（設定から変更推奨）
3. 掛け金・締め切りを設定
4. 大会終了後、結果を入力 → 自動で配当計算

## 仕組み（パリミュチュエル）

- 胴元なし、手数料なし
- 各賭け種別のプールを参加者全員で形成
- 的中者がプールを山分け（的中者が少ないほど高配当）
- オッズ = プール総額 ÷ その組み合わせへの賭け総額
